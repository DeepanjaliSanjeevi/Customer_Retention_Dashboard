# 📊 Customer Churn Analysis Project

## 📌 Project Overview
This project focuses on analyzing customer churn behavior using Excel. The goal is to understand why customers leave, identify retention patterns, and provide actionable business insights to improve customer retention and reduce churn.

---

## 🎯 Objective
The main objectives of this analysis are:
- Understand customer churn patterns
- Identify key factors affecting customer retention
- Analyze churn based on contract type and customer tenure
- Provide insights to improve business decision-making

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Visualization (Charts & Dashboards)  

---

## 📈 Dashboard Overview
An interactive Excel dashboard was created to visualize key metrics such as:
- Customer churn distribution  
- Churn based on contract type  
- Churn across customer tenure  
- Overall retention trends  

---

## 🔍 Key Insights
- Approximately **26% of customers have churned**, indicating significant customer loss.  
- Customers with **month-to-month contracts show the highest churn rate**.  
- Customers with **long-term contracts (1–2 years) demonstrate better retention**.  
- Most churn occurs within the **first 12 months**, indicating early-stage customer drop-off.  

---

## 💡 Recommendations
- Encourage customers to choose **long-term contracts** through discounts or promotional offers.  
- Improve the **initial customer experience (first 3–6 months)** to reduce early churn.  
- Introduce **loyalty programs and incentives** to improve retention.  
- Focus on **high-risk segments (especially month-to-month users)** with targeted retention strategies.  

---

## 🧾 Conclusion
This analysis highlights critical churn patterns and customer behavior trends. By focusing on improving early engagement and encouraging long-term subscriptions, the business can significantly reduce churn and improve overall customer lifetime value and growth.
